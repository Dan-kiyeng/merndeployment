HTML as One of My Core Skills

What is HTML?

HTML stands for HyperText Markup Language. It is used to structure content on the web using tags and elements.

Structural Elements

Examples: html, head, body, section, div.

They define the structure of the web document.

Text Formatting Elements

Examples: h1 to h6, p, b, i, strong, em.

They are used to format and organize text content.

Lists

Examples: ul, ol, li.

They are used to display lists of items.

Links

The a tag is used to create hyperlinks to other web pages.

Images

The img tag is used to embed images in a web page.

Tables

Tags: table, tr, th, td.

Tables are used to display data in rows and columns.

Forms

Tags: form, input, textarea, label.

Forms are used to collect user input.

Semantic Elements

Examples: section, article, header, footer.

They give meaning to the content for browsers and search engines.
